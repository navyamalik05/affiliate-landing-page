COMMON ISSUES & SOLUTIONS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

ISSUE: Site shows 404 error
SOLUTION: 
  1. Check Settings → Pages
  2. Verify source is "Deploy from a branch"
  3. Verify branch is "main"
  4. Wait 2-3 minutes

ISSUE: Changes not showing up
SOLUTION:
  1. Clear browser cache (Ctrl+Shift+Delete)
  2. Hard refresh (Ctrl+Shift+R)
  3. Wait 2-3 minutes
  4. Make sure you committed changes

ISSUE: Video won't play
SOLUTION:
  1. Check YouTube URL is correct
  2. Verify video is public (not unlisted)
  3. Try different embed method

ISSUE: Can't upload files to GitHub
SOLUTION:
  1. Ensure repository is public
  2. Check file names (no spaces)
  3. Try uploading in smaller batches
